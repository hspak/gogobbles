## What is it?

Zero authentication todo "service".
Consists of 2 pages:

    /

For overview and cool things.

    /list/:label

To add/remove todo items from any arbitrary list.

## API

    /api/count
    /api/get/:label
    /api/add/:label/:todo
    /api/remove/:label/:id
